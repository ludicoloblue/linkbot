# Discord Reaction Role Bot

Deploy to Render:
1. Fork this repository
2. Go to render.com
3. Create new Web Service
4. Connect your GitHub repository
5. Set build command: `npm install`
6. Set start command: `node bot.js`
7. Add environment variable: `TOKEN=your_bot_token_here`
8. Deploy

Commands:
- `!reactionroles` - Creates reaction role message
- `!reactionmenu` - Creates dropdown menu for roles
